# 股票分析系统 - 数据导入与交易信号分析

## 功能说明

本项目包含完整的股票数据导入和交易信号分析功能。

### 主要模块

#### 1. **StockDataImporter** - 数据导入器
- 加载CSV格式的股票数据
- 数据格式化和清理
- 数据有效性检查

#### 2. **TradingSignalAnalyzer** - 交易信号分析器
- 计算多种技术指标（SMA、EMA、RSI、MACD、布林线）
- 生成买入信号
- 生成卖出信号
- 计算策略表现

### 技术指标说明

#### 买入信号策略
1. **价格突破20日均线向上** - 上升趋势形成
2. **RSI < 30** - 股票处于超卖状态，可能反弹
3. **MACD金叉** - 快速线穿过慢速线，买入信号
4. **价格触及布林线下轨** - 股票可能反弹

#### 卖出信号策略
1. **价格跌破20日均线向下** - 下降趋势形成
2. **RSI > 70** - 股票处于超买状态，可能回调
3. **MACD死叉** - 快速线穿过慢速线，卖出信号
4. **价格触及布林线上轨** - 股票可能回落
5. **止损** - 跌幅超过2%自动止损

### 技术指标详解

| 指标 | 说明 | 买入信号 | 卖出信号 |
|------|------|--------|--------|
| SMA | 简单移动平均线 | 价格>SMA | 价格<SMA |
| EMA | 指数移动平均线 | 快速EMA>慢速EMA | 快速EMA<慢速EMA |
| RSI | 相对强弱指数 | <30（超卖） | >70（超买） |
| MACD | 移动平均收敛散离 | 金叉 | 死叉 |
| 布林线 | 价格波动带 | 价格<下轨 | 价格>上轨 |

### 输出文件

执行分析后会生成以下文件：

1. **trading_signals.csv** - 所有交易信号（买入/卖出）
2. **data_with_indicators.csv** - 原始数据加上计算的技术指标
3. **trade_records.csv** - 交易记录和利润统计

### 使用方法

```bash
# 运行分析
python data_import.py
```

### 性能指标解释

- **总交易数**: 完整的买入-卖出配对交易数
- **赢利交易**: 盈利的交易数
- **亏损交易**: 亏损的交易数
- **胜率**: 盈利交易占总交易的比例
- **总利润**: 所有交易的累计利润
- **平均收益率**: 平均每笔交易的收益率

### 数据格式要求

CSV文件必须包含以下列（不区分大小写）：
- `date` - 交易日期 (YYYY-MM-DD格式)
- `open` - 开盘价
- `high` - 最高价
- `low` - 最低价
- `close` - 收盘价
- `volume` - 成交量

### 注意事项

1. 至少需要50天的历史数据才能生成可靠信号
2. 技术指标需要预热期，前期信号可能不准确
3. 本分析仅供参考，不构成投资建议
4. 建议结合基本面分析和其他指标共同决策
